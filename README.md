# Geoquiz

### Flutter pet project.

## About the project

| State manager   | Bloc                                       |
|-----------------|--------------------------------------------|
| Auth service    | Firebase                                   |
| Backend service | [airtable.com](https://airtable.com/)      |
| Design          | Neobrutalism                               |
| Platform        | Mobile only                                |
| HTTP Client     | [Dio](https://github.com/flutterchina/dio) |
